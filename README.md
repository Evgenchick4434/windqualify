# windqualify
Проект Зарубина Е. И. для регионального этапа всероссийской олимпиады школьников по технологии, проектного тура 2025-2026 учебного года. 

Для развертывания у себя:

Установите Python с https://www.python.org/downloads/

```bash
git clone https://github.com/Evgenchick4434/windqualify
```
Откройте config.py и вставьте свой токен Telegram-бота и API-ключ метеорологического сервиса

```bash
pip install -r requirements.txt
```
Включите VPN для стабильного подключения к Telegram

```bash
python bot.py
```
